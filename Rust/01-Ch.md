### What is Rust?

- Rust is a **compiled** programming language
    
- Code is **built first**, then **run**
    
- Rust focuses on:
    
    - speed
        
    - safety
        
    - no crashes from memory bugs
        

---

### Compile vs Run

- **Compile** = convert code â†’ machine program
    
- **Run** = execute the program
    

Rust:

`rustc main.rs   # compile ./main          # run`

Cargo shortcut:

`cargo run       # compile + run`

- Curly braces `{}` contain code

### Variables

`let x = 5;`

- `let` = create variable
    
- By default â†’ **cannot change**
    

---

### Mutability

`let mut x = 5; x = 10;`

- `mut` = allow change
    
- Without `mut`, change is **not allowed**
    

---

### Shadowing (IMPORTANT)

`let x = 5; let x = x + 1;`

- This creates a **NEW variable**
    
- Old `x` is gone
    
- Name reused, value replaced safely
    

Rule:

- `mut` â†’ change inside
    
- `let x =` â†’ create new
